<Step>
<StepItem title="Perancangan Antarmuka Pengguna>

  Perancangan antar muka pegguna sangat penting di karenakan untuk membuat sesuatu membutuhkan sebuah
  perancangan yg dimana di matkul ini bukan hanya tentang pemilihan sebuah website agar tampilan nya bagus
  tetapi juga membutuhkan sebuah alur yg jelas seperti flowchart, UML & diagram use case.
  
</StepItem>

<StepItem title="Basis Data">

  Basis data (database) adalah komponen utama dalam pengembangan website Carmart, karena berfungsi
  untuk menyimpan, mengelola, dan mengakses data yang dibutuhkan oleh platform.

</StepItem>

<StepItem title="Praktik Analisi & Desain Perangkat Lunak">

  - Praktik Analisi dan Desain perangkat lunak berhubungan karna untuk membuat sebuah website di
  butuhkan sketsa, wireframe dan prototype nya agar website terintegrasi dengan bagus.

</StepItem>

<StepItem title="Pemrograman Web Lanjut"

  - Berhubungan dengan mendukung pengembangan antarmuka pengguna yang responsif dan interaktif.

</StepItem>

<StepItem title="Metode dan Model Pengembangan Perangkat Lunak">

  - Berhubungan karna mempunyai metode - metode dan model untuk setiap perkembangan perangkat lunak. 
  Metode ini memastikan Carmart dikembangkan dengan fleksibilitas, efisiensi, dan berorientasi pada kebutuhan pengguna.

</StepItem>
