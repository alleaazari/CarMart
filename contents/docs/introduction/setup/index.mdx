 ## Pendauluan dan penjelasan singkat tentang CatMart
 Carmart adalah website rental mobil yang memudahkan pengguna dalam menyewa mobil sport secara online. 
 Proses pembayaran dilakukan melalui WhatsApp (WA)
untuk mempermudah komunikasi langsung dengan admin. Website ini memiliki dua jenis akun, yaitu Admin dan User, di mana setiap user bisa menjadi admin dan sebaliknya.

## Alur Penggunaan Website

### Registrasi dan Login
- User mengunjungi halaman utama Carmart.

- Klik tombol Daftar untuk membuat akun baru.

- Isi formulir registrasi dengan data berikut:

- Nama lengkap

- Email

- Nomor WhatsApp

- Kata sandi

## Melihat Daftar Mobil

Setelah login, user akan diarahkan ke halaman utama yang menampilkan daftar mobil yang tersedia untuk disewa.

User dapat menggunakan fitur pencarian untuk menemukan mobil sesuai kebutuhan (misal: merek, harga, tahun).

## Pilih mobil yang ingin disewa.

- Klik tombol Rentak Sekarang.

- User akan diarahkan ke WhatsApp admin untuk pembayaran dan melakukan rental
dengan mengisi format.

Isi formulir pemesanan dengan detail seperti:

A.Tanggal mulai sewa

B.Tanggal selesai sewa

C.Lokasi penjemputan

D.Klik tombol Kirim Pesanan.

## Mengelola Mobil (Admin)

### Admin login ke dashboard admin.

### Klik menu Kelola Mobil.

### Admin dapat:

- Menambah mobil baru dengan mengisi detail mobil (merek, model, harga, deskripsi, gambar).

- Mengedit informasi mobil yang sudah ada.

- Menghapus mobil dari daftar.

## Mengelola Akun

- Admin dapat masuk ke menu Pengguna di dashboard dengan cara log out menjadi admin dan login lagi sebagai user.
- Maupun admin atau user bisa update atau edit profile.
