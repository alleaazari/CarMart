## Business Requirement
Perusahaan ingin membangun platform website rental mobil online yang memungkinkan pengguna untuk mencari, menyewa, dan merentalkan mobil mereka sendiri. 
Platform ini harus mudah digunakan oleh pengguna, 
baik untuk mencari mobil yang tersedia maupun untuk merentalkan mobil mereka, serta memberikan pengalaman yang nyaman dan aman dalam proses pembayaran melalui WhatsApp.

**Tujuan utama** :
- Menyediakan platform online untuk menyewa mobil dengan proses yang mudah dan cepat.
- Memfasilitasi komunikasi antara penyewa dan penyedia layanan melalui WhatsApp untuk memudahkan transaksi.
- Menyediakan akses mudah ke berbagai mobil yang tersedia untuk disewa, termasuk fitur pencarian berdasarkan merek, harga, dan lokasi.

| Kebutuhan Bisnis                     | Penjelasan                                                                                                       |
|--------------------------------------|------------------------------------------------------------------------------------------------------------------|
| **Fitur Utama Platform**             | Dashboard yang memungkinkan mereka melihat mobil yang telah mereka tambahkan untuk disewakan dan status pesanan  |
| **Integrasi dengan WhatsApp**        | Mengintegrasikan platform dengan WhatsApp untuk memudahkan proses                                                |
|                                      | pembayaran dan komunikasi antara pengguna dan admin.                                                             |

## Manfaat Bisnis (Business Value)

### Menjelaskan nilai yang akan diperoleh perusahaan setelah implementasi sistem:

- *a) Manfaat untuk Perusahaan*

Meningkatkan reputasi sebagai platform rental mobil online yang terpercaya dengan komunitas aktif.

Memperoleh pendapatan tambahan dari iklan promosi dan kemitraan dengan perusahaan terkait.

b) Manfaat untuk Pengguna

Memberikan akses mudah ke berbagai pilihan mobil untuk disewa dengan proses pemesanan yang cepat dan praktis.

Meningkatkan keterlibatan pengguna melalui fitur interaktif seperti notifikasi pesanan dan komunikasi langsung dengan admin melalui WhatsApp.

c) Manfaat untuk Pemilik Mobil (Penjual Mobil)

Memberikan platform untuk merentalkan mobil mereka sendiri dan mendapatkan penghasilan tambahan.

Memberikan pengakuan dan ulasan dari komunitas untuk meningkatkan kepercayaan dari calon penyewa.


## Stakeholder Requirement

**Stakeholder utama** :
- **Penyewa (Customer)**: Pengguna yang ingin menyewa mobil.
- **Penyedia Layanan (Admin)**: Pengelola website yang mengelola daftar mobil dan transaksi.
- **Tim Dukungan Pelanggan**: Tim yang menangani pertanyaan dan masalah pengguna.

## Kebutuhan Stakeholder:

**Penyewa**:
- Kemudahan dalam mencari dan menyewa mobil.
- Proses pembayaran yang aman dan mudah.
- Akses ke informasi yang jelas tentang mobil yang disewa.

**Penyedia Layanan**:
Kemudahan dalam mengelola daftar mobil dan transaksi.
Alat untuk memantau dan menganalisis penyewaan.

## System Requirement

| Kebutuhan Sistem                     | Penjelasan                                                                                                       |
|--------------------------------------|------------------------------------------------------------------------------------------------------------------|
| **Platform web**                     | Website yang responsif dan dapat diakses dari berbagai perangkat (desktop, tablet, smartphone).                  |
| **Database**                         | Sistem manajemen basis data untuk menyimpan informasi pengguna, mobil, dan transaksi.                            |
| **Hubungkan ke WhatsApp**            | Sistem harus terintegrasi dengan WhatsApp untuk memfasilitasi komunikasi dan pembayaran.                         |
| **Keamanan**                         | Sistem harus memiliki fitur keamanan untuk melindungi data pengguna dan transaksi.                               |
| **Backup dan pemulihan**             |  Sistem harus memiliki mekanisme untuk backup data dan pemulihan jika terjadi kegagalan.                         |

## Software Requirement

| Kebutuhan Software                   | Penjelasan                                                                                                       |
|--------------------------------------|------------------------------------------------------------------------------------------------------------------|
| **Bahasa Pemrograman**               | PHP atau Python untuk pengembangan backend.                                                                      |
| **Framework**                        | Laravel (PHP) atau Django (Python) untuk pengembangan aplikasi web.                                              |
| **Database Management System**       | MySQL atau PostgreSQL untuk penyimpanan data.                                                                    |
| **Frontend**                         | HTML, CSS, dan JavaScript (dengan framework seperti Bootstrap atau Vue.js) untuk antarmuka pengguna.             |
| **API WhatsApp**                     | Menggunakan API WhatsApp Business untuk mengelola komunikasi dan pembayaran.                                     |
